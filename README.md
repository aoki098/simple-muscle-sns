# シンプル筋トレSNS

## アプリケーションの概要
- 日々の筋トレのシンプルな記録（種目や最高重量）と、食事のPFCバランスの記録、共有できるフィットネスを専門としたSNSアプリケーションです。
- 記録だけではなく、いいねやコメントなどのユーザー間のコミュニケーション機能や、承認制の鍵垢機能を実装し、安心して記録を共有できるクローズドなコミュニティ設計にこだわりました。
- 日々の記録をSNSに投稿することによって得られる趣味の筋トレのモチベーション維持と、自身の成長記録を残せる自分が使いやすい環境を作りたかったという動機からこのWebアプリを開発しました。

アプリURL（実際のアプリはこちらから動作確認できます）:  https://nextjs-boilerplate-phi-henna-49.vercel.app/

## 使用技術
- フロントエンド: Next.js, React, Tailwind CSS, Lucide React
- バックエンド: Supabase (Authentication, DB, Storage)
- ホスティング: Vercel

## 主な機能
   - サインアップ、ログイン
   - フォロー、ユーザー検索機能
   - おすすめタブ（最新の投稿からランダムに抽出しトップページに表示）、フォロー中タブ（自分がフォローしているユーザーのみの記録を表示）
   - アカウントの鍵垢機能
   - プロフィール編集機能（アイコン、名前、身長体重、自己紹介）
   - 投稿の新規作成、削除、画像投稿機能
   - 投稿に対するいいね、コメント機能

## 使い方
1. 「新規登録」からユーザー登録（ユーザー登録済の場合は「ログイン」からログイン、またはテストアカウントでログイン）
2. 下部メニューの「設定」から、プロフィール（アイコン、名前、自己紹介、身長体重）を編集し、必要な場合アカウントを「鍵垢」に設定します。
3. 下部メニューの「投稿」から、その日の筋トレメニュー（種目・重量）や食事のPFCバランス、写真を記録して投稿します。
4. ホームページの「おすすめ」から新しいユーザーの投稿を発見したり、右上の「検索」から気になるユーザーを探してフォローします（相手が非公開アカウントの場合は承認リクエストが送信され、承認されればフォローできます）。
5. フォローしたユーザーの投稿はホームページの「フォロー中」に表示されます。
   
## 実装上の工夫
 -開発において最も意識したのは、「いかに私のような筋トレ初心者のライト層が挫折せずに筋トレを続け、また使い続けられるアプリにするか」という徹底したユーザー目線です。
  私自身、最近筋トレを始めたばかりの初心者です。既存のアプリのように細かなセット数や回数を毎回入力するのは手間で、ストレスなくアプリを継続して使うのが難しかったです。そこで「私は最大重量が更新されることが一番のモチベーションになる」という自身の体験に基づき、入力項目を極限までシンプルに絞り込むことでストレスなく筋トレの記録ができるアプリにしました。また、筋トレを趣味として長く続けるには他者からの反応が不可欠だと考え、投稿に対する「いいね」や「コメント」機能を実装しました。さらにタイムラインで他のユーザーの投稿（種目や重量、食事内容）を閲覧できるように設計することによりモチベーションの向上だけでなく、新しいトレーニング方法や食事管理の勉強に繋がるようにもしました。
