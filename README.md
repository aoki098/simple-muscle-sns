# シンプル筋トレSNS

## アプリケーションの概要
日々の筋トレのシンプルな記録（種目や最高重量）と、食事のPFCバランスの記録，共有できるフィットネスを専門としたSNSアプリケーションです。
記録だけではなく、いいねやコメントなどのユーザー間のコミュニケーション機能や、承認制の鍵垢機能を実装し、安心して記録を共有できるクローズドなコミュニティ設計にこだわりました。

**▼ 実際のアプリはこちらから動作確認できます**
- アプリURL:https://nextjs-boilerplate-phi-henna-49.vercel.app/

## 使用技術
- フロントエンド: Next.js, React, Tailwind CSS, Lucide React
- バックエンド: Supabase (Authentication, DB, Storage)
- ホスティング: Vercel

## 主な機能
   - サインアップ，ログイン。
   - フォロー，ユーザー検索機能。
   - おすすめタブ（最新の投稿からランダムに抽出しトップページに表示），フォロー中タブ（自分がフォローしているユーザーのみの記録を表示）
   - アカウントの鍵垢機能。
   - プロフィール編集機能（アイコン，名前，身長体重，自己紹介）。
   - 投稿の新規作成，削除，画像投稿機能。
   - 投稿に対するいいね，コメント機能。

## 🔥 開発において最もこだわったポイント
**「UX（ユーザー体験）の細部へのこだわりと、フロントエンドでのデータ保護」**です。

特にタイムラインの「おすすめタブ」の実装においては、単にデータベースから全件取得するのではなく、「非公開アカウント（鍵垢）の投稿を確実に除外する処理」や、「ランダムにシャッフルしつつ、最終的な表示は時系列（新しい順）に並べ直すアルゴリズム」を組み込み、安全性とSNSとしての「新しい出会い」を両立させました。

また、面接官の方や初めてアプリを触る方が迷わないよう、ログイン画面に「ワンタップログイン機能」を実装するなど、ホスピタリティを意識した設計を行っています。
